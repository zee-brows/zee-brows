# Zee Brows

Premium Next.js website and admin dashboard for Zee Brows.

- Public website: `/`
- Admin dashboard: `/admin`
- Admin login: `/admin/login`

See [DEPLOYMENT.md](./DEPLOYMENT.md) for Vercel and Supabase setup.

## Local Development

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3010
```

When using `start-dev.cmd`, the app runs on port `3010`.

## Production Build

```bash
npm run build
```
